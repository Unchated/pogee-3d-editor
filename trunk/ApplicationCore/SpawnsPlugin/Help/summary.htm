Spawns Plugin

The spawns plugin currently provides a means for editing of the spawn2 database table within a three dimensional environment. This plugin is properly installed if a ribbon tab labeled "Spawns" is present in the application.

Usage

If the connection to a database is correctly configured then spawn2 data will be populated automatically when the "Zone" field is filled out within the ribbon tab. If the database connection cannot be established then the working state will be considered offline. In the offline state, any data that is normally pulled from the database (next table id fields, etc) will be improvised and will possibly clash with a populated database if the query is later performed.

Currently only read access is required for database operations, there is currently not an option to commit generated queries directly to the database, primarily for safety and testing reasons.

Controls within the 3D environment

LMB - select closest spawn

LSHIFT+LMB - a new spawn will be created at the location clicked

LCTRL+LMB - with a spawn selected this will cause the spawn selected to face the clicked direction

LALT+LMB - with a spawn selected this will move the spawn to the location clicked
